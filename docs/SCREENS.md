## 営業担当画面（`/sales`）
- KPIごとに大きなカード + `＋KPI名` ボタン。押した瞬間に +1（楽観的更新）、確認画面なし。
- 押下時にカウントのポップ演出・ボタンのフラッシュで視覚フィードバック。
- 各KPIに「現在件数 / 今日の目標 / 達成率 / 進捗バー」。
- KPIボタンの下に「本日の転換率」（分子KPI÷分母KPI）をタップと同時にリアルタイム更新表示。
- 画面下部に固定の「↩ 直前の入力を取り消す（Undo）」。
- 会場セレクタ（選択は端末に保存）。スマホ片手操作前提の大きなタップ領域。

## ダッシュボード（`/dashboard`）
`kpi:update` 受信で自動再取得（+30秒ポーリングの保険）。「リアルタイム更新中」インジケータ表示。

**① デイリー**：本日の全体件数 / 転換率 / 営業別ランキング / 各KPI達成率 / 時間帯別推移（棒グラフ）
**② 当月**：当月累計 / 転換率 / 各ランキング / 各KPI達成率 / 進捗バー

ランキングは主要KPI（表示順が最大＝契約）順に並び、他KPIの内訳をチップ表示。責任者は自部署、管理者は全社。

## 共通
- レスポンシブ: PCは上部ナビ、スマホは下部タブナビに自動切替。
- ダークモード対応（トグルで即時切替、設定は保存）。
- Apple/Notion 風の余白を活かしたデザイン、操作回数を最小化。
